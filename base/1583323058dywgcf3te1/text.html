1. sudo - повышение привелегий до рута на одну команду

2. sudo apt upgrade - обновление приложений

3. sudo apt install openssh-server - установка прриложений, в данном случае ssh

4. ip addr - узнать совй адрес

5. mkdir - создать каталог

6. touch - создать файл

7. rm - удалить файл

8. rm --help - помощь по любой команде

9. pwd - показать текущую директорию (папку, где ты находишься)

10. cat filename - просмотреть файл (как я понял любого формата)

cat > filename - записать в файл то, что введешь с терминала. Или создать файл.

cat file1 file2 > file3 - объединить файлы

11. ctrl + z - свернуть приложение, которое открыл ранее, fg - вернуть обратно

12. ps aux - показать все процессы, kill -15 idпроцесса - предложить процессу завершиться, kill -9 id - убивает процесс нафик


id - посмотреть в каких группах доступа состоишь

chage параметры УЗ - посмотреть параметры УЗ
