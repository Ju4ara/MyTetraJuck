В командной строке необходимо написать следующее:


"C:\Program Files\Oracle\VirtualBox\VBoxManage" modifyhd "f:\Windows server 2008.vdi" --resize 50000


где указываем полный путь к VBoxManage, если есть пробелы, в кавычках

указываем полный путь к жесткому диску, если есть пробелы, в кавычках

размер жесткого диска в мегабайтах


В данном примере мы увеличивыаем размер жесткого диска, который хранится в корне диска F, до 50 Гб
