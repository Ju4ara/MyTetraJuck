Когда настраиваешь не первый раз, изменения следующие:

1. Ставишь Git

2. Заходишь в нужную папку и выполняешь git clone https://Ju4ara@github.com/Ju4ara/MyTetraJuck.git

3. После этого в настройках Тетры прописываешь параметр синхронизации cd e:\ & cd "%a" & git add . & git commit -a -m MyTetraCommit & git pull -s recursive & git push


При первой синхронизации попросит авторизовать устройство
